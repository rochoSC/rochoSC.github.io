DeStress, a smart email suggestion based on current stress levels

Created an Android app and a back-end service using Flask, a Python-based framework which determined email’s stressfulness based on sentiment analysis and adapted UI dynamically to show users which emails they should reply to based on current stress levels measured via a wearable sensor.

Heart rate estimation through accelerometer

Created a dynamic regression model on MATLAB based on accelerometer information for heart rate prediction when exercising. The aim was to reduce energy consumption from smartwatches when using PPG by turning off the sensor and using the prediction model instead.

Built a relaxation app for Android that controlled a robot whose movements were influenced by the user’s breathing obtained through a Bluetooth wearable sensor. The app taught the users to breathe at a proper rhythm to reduce their stress levels.

User-friendly platform for motivational video recording

Created a web app using HTML5, JS, Bootstrap, and Flask. The website had the goal of helping users to record video testimonials through a user-friendly interface and to share them with other users.
